# 1. project idea
The Travel Currency Converter is a web application designed for travelers who want to quickly understand how their money translates across multiple countries at once.

Unlike basic converters, this app allows users to select multiple target currencies simultaneously, helping them compare exchange values in real time and make better financial decisions while traveling

# 2. Purpose
Help users compare currency values across different countries
Provide real-time and historical exchange rate insights
Demonstrate API integration and advanced UI features
Build a user-friendly financial tool for travelers


# 3. API Used
https://frankfurter.dev/

# 4. planning
💱 Multi-Currency Conversion – Convert one base currency into multiple target currencies at once
🔄 Base Currency Toggle – Easily switch between currencies (e.g., USD → INR or INR → EUR)
🕒 Last Updated Timestamp – Shows when exchange rates were last refreshed
📊 7-Day Historical Trend – Displays a sparkline (CSS-based mini chart) of exchange rate changes
🎯 Accurate Real-Time Rates – Uses reliable API data
🌍 Flag Icons – Visual representation of currencies for better UX



# 5. Technologies Used
Frontend: HTML, CSS, JavaScript / React (Vite)
API Integration: Fetch API / Axios
Data Visualization: CSS (for sparkline charts)
Tools: Git, GitHub, VS Code
